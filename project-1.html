TAG - Ticket And Goods

내한공연의 티켓 예매와 오픈마켓 형식으로 공연 관련 상품을 판매하고 구매할 수 있는 쇼핑몰 웹 사이트입니다.

프로젝트 포인트

(위의 이미지를 클릭하면 홈페이지로 이동합니다.)

TAG 웹사이트는 회원이 일반회원과 판매자회원 두가지로 나누어져 있습니다. 일반 공연예매 사이트와 달리 공연 예매 뿐만 아니라 관련 공연의 굿즈 상품을 판매 또는 구매할 수 있으며, 관련 공연을 구매한 회원에겐 굿즈를 구매할 때 포인트와 쿠폰이 발급됩니다.

개발일정

2022.11.17 - 2022.12.27 (총 5주)

  • • 1week: 주제선정, Database 테이블 설계
  • • 2week: UI 가이드맵, 개발 환경 설정
  • • 3week: Front-end, Back-end
  • • 4week: Front-end, Back-end
  • • 5week: 최종 테스트 및 수정, 호스팅

개발환경

  • • Service OS : MacOS, WindowOS
  • • Language : JAVA 1.8
  • • WAS : Apache Tomcat 9.0.58
  • • JDK : 17
  • • DBMS : MariaDB (My SQL)
  • • Spring Boot : 2.7.5, maven, My Batis
  • • Web Hosting : cafe24

본인 담당페이지

  • • 일반 / 판매자 회원가입 및 로그인
  • • 공연 리스트 / 공연 상세
  • • 상품 리스트
  • • 장바구니

티켓 예매

  • • 공연페이지에서는 해당 공연의 상세 정보를 확인할 수 있으며 공연 날짜를 보기 쉽도록 캘린더를 추가했습니다.
  • • 선예매 기간일 경우 별도로 선예매 기간 문구를 노출시켜 사용자에게 정보를 제공합니다.
  • • 공연예매는 실제 블루스퀘어 마스터홀 좌석과 동일하게 구현했으며 좌석선택 → 할인선택 → 수령방법 순서로 진행됩니다.
  • • 공연예매자는 관련 굿즈 할인쿠폰이 발급됩니다.

굿즈 구매

  • • 공연을 예매한 회원은 관련 굿즈를 구매할 수 있는 할인쿠폰이 발급됩니다.
  • • 일반회원은 상품을 장바구니에 담을 수 있으며 바로구매하기가 가능합니다.
  • • 결제 시 관련 공연 굿즈 할인쿠폰을 사용할 수 있습니다.
  • • 그 외 부가기능으로 상품에 좋아요를 누를 수 있습니다.

일반회원 마이페이지

  • • 일반회원은 보유 쿠폰과 포인트, 티켓 예매내역과 구매내역을 확인할 수 있습니다.
  • • 마이페이지에서 티켓예매 취소가 가능합니다.
  • • 좋아요를 누른 공연과 상품을 확인할 수 있습니다.

판매자 마이페이지

  • • 판매자회원은 공연 관련 굿즈를 판매할 수 있습니다.
  • • 마이페이지에서 총 주문건수와 배송상태,미답변문의를 확인할 수 있습니다.
  • • 상품을 등록하고 관리할 수 있습니다.
  • • 상품의 주문을 확인하고 배송상태를 변경할 수 있습니다.
  • • 문의답변을 등록할 수 있습니다.
